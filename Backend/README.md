# Backend - Clasificador de Frutas (DatofrutDiab)

Este módulo contiene la API construida con **FastAPI** y la lógica en Python para el procesamiento digital de imágenes, extracción de características y el entrenamiento del modelo de Inteligencia Artificial utilizando Machine Learning.

## 📋 Requisitos e Instalación
El proyecto cuenta con todas las dependencias y versiones exactas congeladas. Para instalar el entorno correctamente, ejecuta en tu terminal:
```bash
pip install -r requirements.txt

```
# Para levantar el servidor local de FastAPI con recarga automática y permitir que la aplicación móvil de Android se conecte a los endpoints, ejecuta:
```
uvicorn main:app --reload
